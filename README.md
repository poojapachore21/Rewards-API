# Rewards API

This is a Spring Boot-based API for calculating rewards points for customers based on their transactions.

## API Endpoints

- `GET /api/rewards?customerId={customerId}&start={startDate}&end={endDate}`: Get reward points for a customer during a specified date range.

## Technologies Used

- Spring Boot
- Java 8
- JPA/Hibernate

## Setup

1. Clone this repository.
2. Run `mvn spring-boot:run` to start the application.
